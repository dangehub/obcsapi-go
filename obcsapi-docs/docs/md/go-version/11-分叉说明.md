## 关于本分支

本仓库是 [恐咖兵糖 (kkbt)](https://github.com/kkbt0) 原版 [obcsapi-go](https://github.com/kkbt0/obcsapi-go) 的二次修改版。保留原项目名称以表对原作者的尊重。

如无特殊需求，建议使用原版。本分支的改动主要是个人偏好和 CI/CD 适配。

---

## 改动列表

### 1. 午夜跨天可选配置

原版设计为：每天凌晨 00:00-03:59 写入的 Memos 自动算作**前一天**（即凌晨 4 点才刷新日期）。

本版改为**默认关闭**，凌晨 0 点准时按新的一天计算。如需恢复原版行为，在 `config.yaml` 添加：

```yaml
delay_daily_rollover_to_4am: true
```

涉及文件：
- `server/obsidian.go`
- `server/command/luaLibApp.go`

### 2. CI/CD

- 添加 GitHub Actions 自动构建 Docker 镜像并推送到 GHCR
- 添加 VitePress 文档自动构建并部署到 GitHub Pages
- 文档 CI 仅 `obcsapi-docs/docs/` 路径变更时触发，不影响主项目构建

### 3. 构建修复

- 升级 Go 版本至 1.23.0
- 修复 Dockerfile 中移除不存在的 `website/` 目录复制

### 4. 前端开发配置

- 调整 Vite dev server 设置
- `.air.toml` 排除 `webdav/` 目录的热重载

### 5. Docker 镜像

本分支的 Docker 镜像发布在 GitHub Container Registry：

```bash
docker pull ghcr.io/dangehub/obcsapi-go:latest
```
